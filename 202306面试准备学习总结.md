#### 1. js中的null和undefined

摘自阮一峰的 https://www.ruanyifeng.com/blog/2014/03/undefined-vs-null.html

```js
Number(undefined) // NaN

5 + undefined // NaN

```

**null表示"没有对象"，即该处不应该有值**

**undefined表示"缺少值"，就是此处应该有一个值，但是还没有定义。**
